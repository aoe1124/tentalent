# 天赋自测网站 - 需求文档

## 📋 项目概述

**项目名称**：天赋自测网站  
**项目类型**：在线心理测试/性格测试工具  
**目标用户**：对自我认知感兴趣的普通用户  
**技术选型**：HTML + CSS + JavaScript（纯前端，无需后端）

---

## 🎯 核心功能

### 1. 首页/欢迎页
- **标题**：清晰展示"天赋自测"标题
- **简介**：简短介绍测试目的和意义
- **开始按钮**：引导用户进入测试
- **设计风格**：简洁、现代、符合Apple审美（大量留白、柔和配色）

### 2. 测试问答页面
- **题目展示**：
  - 一次展示一道题目
  - 题目序号显示（加进度条效果）（例如：第1题/共40题）
  
  
- **交互设计**：
  - 选项采用卡片式设计，点击后高亮（紫色边框）
  - "下一题"按钮（点击选项后自动出现或延迟跳转）
  - "上一题"按钮（可返回修改答案）
  - 流畅的淡入淡出过渡动画
  - 最后一题显示"查看结果"按钮

### 3. 结果展示页
- **标题**：
  - "恭喜完成测试"（居中显示）
  
- **综合评价文字**：
  - 格式："你最具有 [天赋1]、[天赋2]、[天赋3]，然而 [天赋8]、[天赋9]、[天赋10] 相对较弱。"
  - 前3名用紫色高亮显示
  - 后3名用灰色显示
  
- **天赋能力雷达图**：
  - 标题："天赋能力图"
  - 10边形雷达图，每个顶点代表一个维度
  - 显示维度代号（A、B、C...J）和分数
  - 紫色填充区域，带渐变效果
  - 灰色网格背景
  
- **各项天赋分值列表**：
  - 标题："各项天赋分值"
  - 10个天赋维度，每个显示：
    - 序号 + 名称（如"A. 语言："）
    - 紫色进度条（根据分数长度）
    - 分数显示（白色文字在进度条上）
    - 评价文字（右侧，如"有些擅长"、"不太擅长"）
  - 进度条样式：
    - 高分（15-20）：深紫色
    - 中分（12-14）：中紫色
    - 低分（7-11）：浅灰色背景
  
- **操作按钮**：
  - "返回首页"按钮（灰色）
  - "重新测试"按钮（紫色渐变）
  - 两个按钮并排显示，圆角设计

## 🎨 设计要求

### 设计理念

**核心原则**：专业、高级、克制  
**设计方向**：移动端优先（Mobile First）+ 精致简约主义

#### 竞品问题分析
通过分析竞品设计，发现以下可改进点：
1. ❌ 选项卡片间距略挤，呼吸感不足
2. ❌ 字体层级不够清晰，排版较普通
3. ❌ 进度条设计过于简单，缺乏精致感
4. ❌ 整体视觉层次感欠佳
5. ❌ 缺乏品质感的细节处理

#### 我们的解决方案
1. ✅ **更大的留白**：移动端卡片间距20px，营造呼吸感
2. ✅ **精致的排版**：使用专业字体系统，层级分明
3. ✅ **高级的配色**：克制的色彩系统，减少饱和度
4. ✅ **细腻的细节**：微妙的阴影、渐变、动画
5. ✅ **流畅的体验**：移动端手势优化，动效流畅

### 移动端体验优化

#### 触摸优化
```
- 所有可点击元素最小触摸区域：44x44px
- 卡片间距不小于16px，避免误触
- 点击反馈：150ms内显示视觉反馈
- 禁用：移动端的hover效果（用active替代）
```

#### 手势优化
```
- 支持左右滑动切换题目（可选功能）
- 下拉刷新：重新开始测试
- 防止误触：选中后500ms延迟再允许下一步
```

#### 性能优化
```
- 使用CSS transform替代position动画
- 图片懒加载
- 减少重绘重排
- 使用will-change提示浏览器优化
```

### 视觉细节提升

#### 微妙细节
```
1. 卡片悬浮层次：使用多层阴影叠加
   box-shadow: 
     0 2px 4px rgba(15, 23, 42, 0.04),
     0 4px 8px rgba(15, 23, 42, 0.06),
     0 8px 16px rgba(15, 23, 42, 0.08);

     circle at 50% 0%,

     rgba(255, 255, 255, 0) 50%
   );

3. 分隔线：不用纯灰色，用带透明度的黑色
   border-color: rgba(15, 23, 42, 0.08);

4. 文字抗锯齿：
   -webkit-font-smoothing: antialiased;
   -moz-osx-font-smoothing: grayscale;
```

### 设计对比总结

| 设计要素 | 竞品 | 我们的方案 | 提升点 |
|---------|------|-----------|-------|
| 卡片间距 | 12-16px | 20px | 呼吸感更好 |
| 圆角 | 8px | 12px | 更柔和高级 |
| 阴影 | 简单阴影 | 多层叠加阴影 | 层次感更强 |
| 字体 | 常规 | 优化间距和粗细 | 更清晰易读 |
| 配色 | 高饱和紫 | 降低饱和度 | 更专业克制 |
| 动画 | 基础过渡 | 精心设计的缓动 | 更流畅自然 |
| 进度条 | 3px | 4px(移动)/6px(桌面) | 更清晰 |
| 触摸区域 | 未优化 | 最小44x44px | 更易操作 |

**总结**：通过克制的色彩、精致的细节、流畅的动画、移动端优先的布局，打造一个比竞品更专业、更高级但不花哨的天赋测试网站。

## 📊 测试逻辑

### 题目数量
- **总题数：40道题**
- 每题4-6个选项
- 每个选项对应不同维度的分数加成

### 天赋维度分类（基于多元智能理论）
共10个天赋维度：

1. **A. 语言天赋** - 语言表达、写作、阅读能力
2. **B. 逻辑-数学天赋** - 逻辑推理、数学计算、科学思维
3. **C. 空间天赋** - 空间想象、方向感、视觉艺术
4. **D. 内省天赋** - 自我认知、情绪管理、反思能力
5. **E. 人际天赋** - 社交沟通、同理心、人际关系
6. **F. 身体-动觉天赋** - 运动协调、手工技能、身体控制
7. **G. 音乐天赋** - 音乐感知、节奏韵律、音乐创作
8. **H. 自然天赋** - 自然观察、动植物认知、环境感知
9. **I. 创造天赋** - 创新思维、想象力、艺术创造
10. **J. 美学天赋** - 审美能力、艺术鉴赏、设计感知

### 题目选项格式
每道题都使用相同的5个选项：
1. **非常符合** - 额外 +2.5分
2. **很符合** - 额外 +2分
3. **部分符合** - 额外 +1.5分
4. **不太符合** - 额外 +1分
5. **完全不符合** - 额外 +0分

### 计分规则
- **题目分配**：40道题，每个维度4道题
- **基础分**：每个维度基础分10分
- **额外加分**：根据答题情况，每题可额外获得0-2.5分
- **分数区间**：每个维度总分 **10-20分**
  - **最低分**：10 + 0×4 = 10分（4题都选"完全不符合"）
  - **最高分**：10 + 2.5×4 = 20分（4题都选"非常符合"）
  
- **评价标准**（参考竞品）：
  - **17-20分**：非常擅长（深紫色进度条，显示"非常擅长"）
  - **14-16分**：有些擅长（中紫色进度条，显示"有些擅长"）
  - **10-13分**：不太擅长（浅灰色进度条，显示"不太擅长"）
- **结果展示逻辑**：
  - 对10个维度按分数排序
  - 显示前3名最高分的天赋（用紫色/粉色高亮显示在开头）
  - 显示后3名较低分的天赋（用灰色显示在结尾）
  - 雷达图展示10个维度的完整分布
  - 横向进度条展示各维度详细分数和评价
