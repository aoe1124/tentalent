# 天赋自测网站

一个基于多元智能理论的在线天赋测试工具，采用移动端优先设计，专业、高级、克制的视觉风格。

## 📋 项目简介

本项目是一个纯前端实现的天赋测试网站，通过40道精心设计的题目，测评用户在10个维度的天赋表现：

- **A. 语言天赋** - 语言表达、写作、阅读能力
- **B. 逻辑-数学天赋** - 逻辑推理、数学计算、科学思维
- **C. 空间天赋** - 空间想象、方向感、视觉艺术
- **D. 内省天赋** - 自我认知、情绪管理、反思能力
- **E. 人际天赋** - 社交沟通、同理心、人际关系
- **F. 身体-动觉天赋** - 运动协调、手工技能、身体控制
- **G. 音乐天赋** - 音乐感知、节奏韵律、音乐创作
- **H. 自然天赋** - 自然观察、动植物认知、环境感知
- **I. 创造天赋** - 创新思维、想象力、艺术创造
- **J. 美学天赋** - 审美能力、艺术鉴赏、设计感知

## ✨ 特色功能

### 设计特点
- 🎨 **专业高级** - Apple风格的极简设计，克制的配色系统
- 📱 **移动端优先** - 优先考虑移动端体验，响应式适配各种屏幕
- 💎 **精致细节** - 多层阴影、流畅动画、考究的排版
- 🎯 **用户友好** - 清晰的进度提示、流畅的交互体验

### 核心功能
- ✅ **40道专业题目** - 基于多元智能理论设计
- ✅ **实时进度保存** - 使用localStorage，刷新不丢失
- ✅ **雷达图可视化** - 直观展示10个维度的能力分布
- ✅ **详细结果分析** - 综合评价 + 分值列表 + 建议
- ✅ **无需后端** - 纯前端实现，可直接打开使用

## 🚀 快速开始

### 方法1：直接打开（推荐）
1. 下载项目文件
2. 用浏览器打开 `index.html` 即可使用

### 方法2：本地服务器
```bash
# 使用Python启动本地服务器
python -m http.server 8000

# 或使用Node.js的http-server
npx http-server
```

然后访问 `http://localhost:8000`

### 方法3：部署到GitHub Pages
1. 上传项目到GitHub仓库
2. 在仓库设置中启用GitHub Pages
3. 选择主分支（main/master）
4. 网站将自动部署

## 🎨 设计规范

### 配色系统
- **品牌紫**：#7C3AED（主色调）
- **中性色**：#0F172A（文字）、#64748B（副文字）、#F1F5F9（背景）
- **渐变**：linear-gradient(135deg, #7C3AED 0%, #A78BFA 100%)

### 字体系统
- **移动端**：16px基础字号
- **桌面端**：17px基础字号
- **字体**：-apple-system, BlinkMacSystemFont, "PingFang SC", "Microsoft YaHei"

### 响应式断点
- **移动端**：320px - 768px
- **平板**：768px - 1024px
- **桌面**：1024px+

## 📊 计分规则

### 选项分值
- 非常符合：+2.5分
- 很符合：+2.0分
- 部分符合：+1.5分
- 不太符合：+1.0分
- 完全不符合：+0分

### 分数计算
- 每个维度基础分：10分
- 每个维度4道题
- 最终分数区间：10-20分

### 评价标准
- **17-20分**：非常擅长（深紫色）
- **14-16分**：有些擅长（中紫色）
- **10-13分**：不太擅长（浅灰色）

## 🔧 自定义修改

### 修改题目
编辑 `js/questions.js` 文件中的 `QUESTIONS` 数组。

### 修改配色
修改 `css/style.css` 中的 CSS变量：
```css
:root {
  --color-primary: #7C3AED;  /* 修改主色调 */
  /* ... 其他颜色变量 */
}
```

### 修改评分规则
修改 `js/result.js` 中的 `calculateScores` 函数。

## 👨‍💻 开发说明

### 设计理念
本项目遵循"专业、高级、克制"的设计理念：
- **专业** = 精准的排版 + 合理的布局 + 清晰的层级
- **高级** = 微妙的细节 + 考究的配色 + 流畅的动效
- **克制** = 不花哨 + 不过度 + 恰到好处

### 移动端优先
- 所有触摸元素最小44x44px
- 卡片间距20px，避免误触
- 使用transform替代position动画（性能更好）
- 禁用移动端双击缩放

### 性能优化
- 使用CSS transform实现动画（GPU加速）
- 雷达图支持高DPI屏幕
- LocalStorage实现进度自动保存
- 无外部依赖，加载速度快
